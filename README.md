Лабораторна робота №2

Апроксимація функцій та класифікація за допомогою нейронних мереж (MATLAB)


 Опис

У даній лабораторній роботі досліджується застосування нейронних мереж для:

- апроксимації функцій
- обробки часових рядів
- розв’язання задач класифікації

Робота виконана в середовищі MATLAB з використанням Neural Network Toolbox.

 Мета роботи
 
- Ознайомитися з адаптивними лінійними мережами (Adaline)
- Дослідити багатошарові нейронні мережі
- Вивчити вплив параметрів навчання
- Реалізувати задачі апроксимації та класифікації

Структура роботи

Частина 1 – Апроксимація функцій

- Експеримент 1 – проста залежність
- Експеримент 2 – гармонічний сигнал
- Експеримент 3 – сигнал зі змінною частотою та затримкою

 Використовується:

- `linearlayer`
- `preparets`
- часові ряди

Частина 2 – Класифікація

XOR

- реалізація нелінійної логічної функції
- використання `feedforwardnet`

Розпізнавання символів

- класифікація векторів (імітація букв)
- дослідження впливу шуму

Результати

- Лінійна мережа ефективна для простих залежностей
- Багатошарова мережа вирішує нелінійні задачі (XOR)
- Зі збільшенням шуму точність зменшується
- Затримка покращує роботу з часовими рядами

Приклади результатів

У проєкті представлені:

- графіки навчання
-  графіки порівняння сигналів
-  Training State
-   структура нейронних мереж

Використані технології

- MATLAB
- Neural Network Toolbox

Як запустити

1. Відкрити MATLAB
2. Завантажити `.m` файл
3. Запустити скрипт
4. Переглянути результати у вигляді графіків

Висновок

Було досліджено ефективність нейронних мереж для задач апроксимації та класифікації. Встановлено, що правильний вибір архітектури та параметрів навчання є ключовим для досягнення високої точності.
